# agentic-assistant

A Discord bot powered by the [agentic](https://github.com/jereanon/agentic) library. Supports Claude and OpenAI models.

## Setup

1. Copy `config.example.toml` to `assistant.toml` and fill in your settings
2. Set environment variables for API keys (`ANTHROPIC_API_KEY`, `DISCORD_TOKEN`, etc.)
3. `cargo run --release`

## Configuration

See [config.example.toml](config.example.toml) for all available options.
